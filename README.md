# リクエストサンプル

リーチ状態（正方形１つ前）
```json
{
    "stone0": {"x": 0, "y": 0},
    "stone1": {"x": 0, "y": 1},
    "stone2": {"x": 1, "y": 0},
    "stone3": {"x": 2, "y": 2}
}
```

積み（正方形）
```json
{
    "stone0": {"x": 0, "y": 0},
    "stone1": {"x": 0, "y": 1},
    "stone2": {"x": 1, "y": 0},
    "stone3": {"x": 1, "y": 1}
}
```
